Willkommen zur Produkt-API

Dies ist eine einfache RESTful API zur Verwaltung von Produkten. Die API verwendet Javalin als Web-Framework und MongoDB als Datenbank.

API-Dokumentation

Die vollständige API-Dokumentation ist über Swagger UI verfügbar:

Swagger UI öffnen

Verfügbare Endpunkte

  • GET /api/products - Alle Produkte abrufen
  • GET /api/products/{id} - Produkt nach ID abrufen
  • POST /api/products - Neues Produkt erstellen
  • PUT /api/products/{id} - Produkt aktualisieren
  • DELETE /api/products/{id} - Produkt löschen
